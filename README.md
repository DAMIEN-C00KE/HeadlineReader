# HeadlineReader
Scrapes headlines from the inputted URLs, and prints them to an SQLite database. (# Work in progress)

Before running, ensure you've installed the required packages via the Julia REPL:
- import Pkg
- Pkg.add("HTTP")
- Pkg.add("Gumbo")
- Pkg.add("Cascadia")
- Pkg.add("SQLite")
- Pkg.add("Dates")
- Pkg.add("JSON")
